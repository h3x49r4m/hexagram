The Quadratic Formula

One of the most fundamental formulas in algebra:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Derivatives

The derivative of a function $f(x)$ is defined as:

$$f’(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

Code Implementation

Here’s a Python function to calculate derivatives numerically:

import numpy as np

def numerical_derivative(f, x, h=1e-5):
    """
    Calculate numerical derivative using central difference method
    
    Parameters:
    f: function
    x: point to evaluate derivative
    h: step size
    
    Returns:
    derivative approximation
    """
    return (f(x + h) - f(x - h)) / (2 * h)

# Example usage
def f(x):
    return x**2 + 3*x + 2

derivative_at_2 = numerical_derivative(f, 2)
print(f"f'(2) ≈ {derivative_at_2}")
